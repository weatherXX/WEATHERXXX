# WEATHERXXX 🌤️

A beautiful real-time weather app built with React + Node.js

## Features
- 🌡️ Real-time weather data (OpenWeatherMap)
- ☀️ UV Index (Tomorrow.io)
- 🌿 Real pollen data (Open-Meteo)
- 🌙 Moon phase + real moonrise/moonset
- 🏃 Smart running score based on real conditions
- 💾 Save multiple cities
- 📍 GPS location detection
- 💨 Wind compass, pressure arc, sun arc

## Setup

### Backend
\`\`\`bash
cd backend
npm install
cp .env.example .env
node src/index.js
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm start
\`\`\`

## APIs Used
- [OpenWeatherMap](https://openweathermap.org/api)
- [Tomorrow.io](https://tomorrow.io)
- [Open-Meteo](https://open-meteo.com)
